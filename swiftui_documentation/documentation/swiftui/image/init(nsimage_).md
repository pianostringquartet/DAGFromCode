# init(nsImage:)

**Available on:** macOS 10.15+

> Creates a SwiftUI image from an AppKit image instance.

```swift
nonisolated init(nsImage: NSImage)
```

## Parameters

**nsImage**

The AppKit image to wrap with a SwiftUI [Image](/documentation/swiftui/image). instance.
