# init(rootView:)

**Available on:** macOS 14.4+

> Creates a hosting menu object that wraps the specified SwiftUI view.

```swift
init(rootView: Content)
```

## Parameters

**rootView**

The root view of the SwiftUI view hierarchy that you want to present using this menu.
