# init(initialValue:)

**Available on:** iOS 13.0+, iPadOS 13.0+, Mac Catalyst 13.0+, macOS 10.15+, tvOS 13.0+, visionOS 1.0+, watchOS 6.0+

> Creates an observed object with an initial value.

```swift
@MainActor @preconcurrency init(initialValue: ObjectType)
```

## Parameters

**initialValue**

An initial value.
